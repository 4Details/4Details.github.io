HyperLoglog

  • 采用一种基数算法，用于完成独立总数的统计。
  • 占据空间小，无论统计多少个数据，只占12K的内存空间。
  • 不精确的统计算法，标准误差为0.81%。

Bitmap

  • 不是一种独立的数据结构，实际上就是字符串。
  • 支持按位存取数据，可以将其看成是byte数组。
  • 适合存储大量的连续的数据的布尔值。

如论坛签到？两种状态，用连续的01字符串存储
